Android Jelly Bean- Features

Read my earlier post about Android Jelly Bean for an introduction

Currently, after the release of the Android Ice cream Sandwich, most of the android users are currently looking forward for the details about the next Android release by Google. Many Tech  Blogs Cites that the next release of android will be named as Android Jelly Bean according to the information from some trusted sources. At the same time many other popular blog authors say that the name has not been finalised yet.  According to rumors, and all previous versions, this one also has roots in desserts/sweets. Eventhough there's no official informations about the features of Android Jelly Bean, heres a list of some possible features of Android Jelly Bean.

Android Jelly Bean Features:

1. Voice activated assistant like SIRI

2.Fully functional Cloud computing

3. A better in built on screen keyboard which is highly customizable

4. In bulilt- Customizable Launcher/ Home screen with support to themes

5. High Efficiency in using battery power

6. Chrome browser giving a Desktop browsing experience

7. In built Task Manager

8. Lock screen Widgets

9. A Complete file manager

10. Long pressing back button kills the App
