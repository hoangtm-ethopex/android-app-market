Call Log In An Android Application

How to use/access Phone Call Log in Android Application?

An Android Application can access the call log of the android phone. This is possible because the call log details are exposed by the Android platform as a content provider. Content Providers in Android allow applications to share data across applications (Refer: http://www.android-app-market.com/content-provider-in-android.html). Numerous inbuilt content providers are there with the Android platform. One such content Provider is the CallLog.Calls content provider.
(Refer: http://developer.android.com/reference/android/provider/CallLog.Calls.html)
