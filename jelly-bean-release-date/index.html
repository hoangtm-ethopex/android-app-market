Android Jelly Bean - Next Android Release After the Icecream Sandwich


 

Check Out The Features of Android Jelly Bean

Rumours have been spreading in the tech world about the next release of Android OS that it will be named as "Jelly Bean". By analysing googles tradition of naming android versions, we can expect that the next release of android will be in the name of a sweet starting with "J". Probably it will be Android Jelly Bean. But no one yet dont have any idea about the version number of the next release. The only thing we know is that the next android version number will be something higher than the version number of Icecream sandwich, that is a number higher than 4.0.
Google's naming tradition of major android releases:
1.5 Cupcake
1.6 Donut
2.0 Eclair
2.2 Froyo
2.3 Gingerbread
3.0 Honeycomb
4.0 Icecream Sandwich
Rumours are also about the key features that google planning in the next release. The "Game Changing Stuff" which was supposed to be available in the Icecream Sandwich might be pushed to the Jelly Bean. But we have no idea about what other features are they going to introduce in the Jelly Bean. We can only expect that the next android release will give much more importance to Gaming.
