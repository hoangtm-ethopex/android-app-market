Project Majel - Google's Answer to Siri

Latest rumors suggests that Google is working on a project codenamed as Majel, which is the rival of Siri. We all know about the cool features of Siri, or at least have heard about some of them. Siri uses your voice to send messages, set reminders, search information on the net and much more. There are many apps in the android market claiming that they possess the features of Siri, but I found none of them worthy. Even before Siri, there were voice command assistants, but listening only for the specific words or syntax. Siri changed the concept and made a story of success.

The project Majel was expected to complete before the end of 2011. But till now, there is no official announcements from google about project Majel. Android fans are eagerly waiting to see what great features are they going to implement with that. I think its worth waiting coz reports suggest that Google engineers are working hard to make something which delivers better performance than Siri.

One way for Google to outsmart Siri is by allowing deeper integration into the Android kernel.  As Siri is not capable of doing complex tasks even now, such as send a tweet, navigational tasks etc. and if Majel can manage to handle these tasks in Android, we can say that Majel will surely outsmart Siri.

It is expected that Majel will find its way to market in 2012 itself. We have no idea about how Google is planning to implement Majel, whether as a stand alone application or integrated to Android kernel. While the next Android release- Android Jelly Bean is about to hit the market in this year, we can expect that the major feature with the Jelly Bean will be the Majel.

Android Jelly Bean May be Featuring a Voice Command Virtual Assistant like SIRI

Also Visit:

Android Jelly Bean- Next Android Release

Features of Android Jelly Bean

Android 5.0 Jelly Bean - According to latest informations, Google is working on the next major release of Android. Android 5.0 Jelly Bean will be an An Android update that contains some of the features that were originally been scheduled for Android 4.0 Icecream sandwich. One of those features is the so called "Game Changing Stuff" which was initially planned to introduce in Icecream Sandwich but no one has any idea about what's this "Game Changing Stuff" is. Only thing we can guess is that this feature will improve the gaming experience to a higher level.

As iOS 5 is released with lot of new features and iPad 3.0 is all set to be released in March, Android is expected to offer much more in this year, than they had done before.  Apple had recently introduced the Voice Command Feature called SIRI in the iPhone 4S. Actually SIRI was an iPhone app available in the App Store for download. They were planning to release Android and Blackberry versions of SIRI. But SIRI was acquired by Apple and with that their efforts to release SIRI on other platforms was cancelled. Now SIRI is an integrated feature of iPhone 4S. Since the version 2.2, Android has an inbuilt Text-to-Speech and Speech-to-Text engine in it. But now Android users are looking for a fully functional Voice Commanding software like SIRI which is able to do whatever SIRI does in iPhone 4S. Inorder to keep in pace with the iOS, google has to introduce a similar feature in Android as fast as possible. If rumours are to be believed, Google is working on a similar feature to be introduced with the next release of Android- Android 5.0 Jelly Bean.

Thus we can expect that the Android Jelly Bean will come with a Voice Activated Virtual Assistant like SIRI, and also with lots of other powerful features.

Android Jelly Bean - Next Android Release After the Icecream Sandwich


 

Check Out The Features of Android Jelly Bean

Rumours have been spreading in the tech world about the next release of Android OS that it will be named as "Jelly Bean". By analysing googles tradition of naming android versions, we can expect that the next release of android will be in the name of a sweet starting with "J". Probably it will be Android Jelly Bean. But no one yet dont have any idea about the version number of the next release. The only thing we know is that the next android version number will be something higher than the version number of Icecream sandwich, that is a number higher than 4.0.
Google's naming tradition of major android releases:
1.5 Cupcake
1.6 Donut
2.0 Eclair
2.2 Froyo
2.3 Gingerbread
3.0 Honeycomb
4.0 Icecream Sandwich
Rumours are also about the key features that google planning in the next release. The "Game Changing Stuff" which was supposed to be available in the Icecream Sandwich might be pushed to the Jelly Bean. But we have no idea about what other features are they going to introduce in the Jelly Bean. We can only expect that the next android release will give much more importance to Gaming.
