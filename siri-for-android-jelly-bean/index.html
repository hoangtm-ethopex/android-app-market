Android Jelly Bean May be Featuring a Voice Command Virtual Assistant like SIRI

Also Visit:

Android Jelly Bean- Next Android Release

Features of Android Jelly Bean

Android 5.0 Jelly Bean - According to latest informations, Google is working on the next major release of Android. Android 5.0 Jelly Bean will be an An Android update that contains some of the features that were originally been scheduled for Android 4.0 Icecream sandwich. One of those features is the so called "Game Changing Stuff" which was initially planned to introduce in Icecream Sandwich but no one has any idea about what's this "Game Changing Stuff" is. Only thing we can guess is that this feature will improve the gaming experience to a higher level.

As iOS 5 is released with lot of new features and iPad 3.0 is all set to be released in March, Android is expected to offer much more in this year, than they had done before.  Apple had recently introduced the Voice Command Feature called SIRI in the iPhone 4S. Actually SIRI was an iPhone app available in the App Store for download. They were planning to release Android and Blackberry versions of SIRI. But SIRI was acquired by Apple and with that their efforts to release SIRI on other platforms was cancelled. Now SIRI is an integrated feature of iPhone 4S. Since the version 2.2, Android has an inbuilt Text-to-Speech and Speech-to-Text engine in it. But now Android users are looking for a fully functional Voice Commanding software like SIRI which is able to do whatever SIRI does in iPhone 4S. Inorder to keep in pace with the iOS, google has to introduce a similar feature in Android as fast as possible. If rumours are to be believed, Google is working on a similar feature to be introduced with the next release of Android- Android 5.0 Jelly Bean.

Thus we can expect that the Android Jelly Bean will come with a Voice Activated Virtual Assistant like SIRI, and also with lots of other powerful features.
