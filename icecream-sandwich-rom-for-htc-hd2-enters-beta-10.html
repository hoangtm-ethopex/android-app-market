Icecream Sandwich ROM for HTC HD2 Enters Beta 10


The Android Icecream sandwich port for HD2 enters beta 10 with almost all essential functions working. The only known issues are with Native USB Tethering and VPN. The XDA- Developers team is making hardwork in porting the icecream sandwich for HD2. It is built from the modified AOSP source code.
The MAGLDR-CWR users can download the ROM from here and install.
Use 190MB system partition and 2MB cache

Working:

  • Phone Calls
  • Audio
  • SMS
  • Data: 3G/HSDPA (RMNET for MAGLDR) or (PPP for cLK)
  • Wi-Fi
  • Bluetooth (in-call works)
  • GPS and AGPS
  • Touchscreen
  • Vibration
  • Charging
  • All sensors
  • Wired earphone/headphone
  • Data usage (i.e. Network Stats and Quota)
  • Hardware Acceleration OpenGL
  • Hardware Acceleration OpenMAX codecs
  • Software video codecs
  • WiFi signal strength icon.
  • USB Mass Storage (Full working)
  • Camera
  • Native WiFi Tethering

Not Working:

  • Native USB Tethering
  • VPN (Not test on 4.0.3)
(Source: XDA-Developers site)

 

You can visit the original XDA-Developers thread from here for more information and changelog
